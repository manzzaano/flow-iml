## Sobre Flow

Flow es un proyecto personal diseñado para perfeccionar el flujo de trabajo profesional en el desarrollo de software. Este simulador permite practicar metodologías de ingeniería avanzada (Scrum, Git Flow, Code Reviews) en un entorno de calma y control, utilizando agentes de IA para replicar la dinámica de un equipo real.

- **Calma Productiva:** Interfaz diseñada en tonos Dark Mint para optimizar el enfoque.
- **Aprendizaje de Procesos:** Simulación de roles profesionales (PO, Scrum Master, Tech Lead).
- **Control de Ingeniería:** Integración con flujos de trabajo de Big Tech adaptados para desarrolladores en crecimiento.
